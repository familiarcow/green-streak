# Changelog

All notable changes to Green Streak will be documented in this file.

## [1.1.0] - 2026-02-13

### Added
- **Goals System** - Set meaningful goals and track progress with visual timelines
  - Create goals with templates or from scratch
  - Add milestones to mark important moments on your journey
  - Primary goal designation for quick access
  - Goal-based habit template sorting
- **Achievement Grid** - New visual achievement display
  - Beautiful grid layout revealing artwork as you unlock achievements
  - Glass-break unlock animations
- **Sound Effects & Haptics** - Rich audio-tactile feedback throughout the app
  - Tap sounds, success chimes, and celebratory effects
  - Haptic feedback for interactions
  - Drag sounds on color picker
- **Expanded Achievements** - Science-backed milestone system with more unlockables
- **24-Hour Time Format** - Option in settings for international users
- **Calendar View Filtering** - Choose which habits appear in the default calendar view
- **App Rating Prompt** - Appears after 3 active days (dismissible)
- **Notification Improvements**
  - Visual preview with animated schedule selector
  - Clock dial time picker
  - Weekly reminder day-of-week selection

### Changed
- **Habit Cards Redesigned** - Drag handle moved to left side for easier reordering
- **Goal Cards Redesigned** - Period selector in header with live progress updates
- **Template Catalog** - New 2-column grid layout with quick-add option
- **Settings Screen** - Collapsible sections, simplified layout, moved Goals to bottom
- **Onboarding Flow** - Added settings step, improved copy, goal integration

### Fixed
- iOS freeze when achievement modal opens immediately after another modal
- Month markers displaying on wrong day due to timezone bug
- Rating prompt showing before app data fully loaded
- Achievement cache not updating after data import
- Template quick-add causing app freeze
- Double close sound on X buttons
- LiveCalendar screen pinning scroll behavior

## [1.0.1] - 2026-02-04

### Changed
- Updated onboarding screen with animated contribution graph demo
- Improved visual hierarchy on onboarding action buttons
- Refined achievement unlock animation for a more professional feel

## [1.0.0] - Initial Release

### Added
- Habit tracking with GitHub-style contribution graphs
- Multiple habit support with custom colors and icons
- Achievement system with unlockable milestones
- iOS widgets for home screen
- Customizable app icons
- Privacy-first design - all data stored locally
