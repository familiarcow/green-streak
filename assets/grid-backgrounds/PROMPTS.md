# Achievement Grid Background Image Prompts

Use these prompts with Nano Banana to generate 12 cozy isometric illustration backgrounds for the achievement grid.

---

## Unified Style Specification

| Attribute | Specification |
|-----------|---------------|
| **Style** | Soft illustration with hand-drawn textures, gentle edges, watercolor-inspired warmth |
| **Perspective** | Isometric view (30° angle), looking down at 45° diagonal |
| **Lighting** | Golden hour - warm amber/peach tones, soft diffused shadows, sun low on horizon |
| **Color Palette** | Warm neutrals (cream, beige, soft white) + muted pastels + accent pops matching app rarity colors |
| **Texture** | Subtle paper grain, soft brush strokes, organic imperfections |
| **Mood** | Cozy, inviting, aspirational, peaceful |
| **Composition** | Centered subject with breathing room, environmental storytelling |
| **Aspect Ratio** | 1:1 square (to match 7x7 grid display) |

### Color Harmony (matching app palette)
- **Primary accents**: Forest green (#22c55e), warm cream (#F8EAC7)
- **Rarity integration**: Gold highlights, purple shadows, blue mid-tones, green foliage
- **Warm base**: Cream/eggshell undertones throughout

### Recommended Generation Settings
- **Resolution**: 1024x1024 or higher (square format)
- **Guidance Scale**: 7-9 (balanced between prompt adherence and creativity)
- **Steps**: 30-50 for detailed illustration quality

## Consistency Checklist

Before generating, verify each prompt includes:

- [x] "Isometric view" perspective
- [x] "Soft illustration watercolor style"
- [x] "Golden hour/morning/afternoon light"
- [x] Warm cream/amber color palette
- [x] "No people" specification
- [x] Cozy, peaceful mood descriptors
- [x] Hand-drawn/storybook style reference
- [x] 1:1 aspect ratio

## Post-Generation Notes

1. **Save images as PNG** for best quality with transparency support
2. **Verify warm tones** - reject any images with cold/blue undertones
3. **Check composition** - important details should be distributed across the image (not just centered) since ~70% will be covered by grid cells
4. **Test in app** - mock up with achievement grid to verify visual harmony with glass effects
